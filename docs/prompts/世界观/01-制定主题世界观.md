# 制定“[主题]”的世界观

> 围绕指定主题建立规则明确、能够产生剧情冲突且适配当前项目的世界观框架。

## 可直接使用的指令

```prompt
请为当前 Storydex 小说项目制定以“[主题]”为核心的世界观方案。

要求：
1. 先读取当前项目的时代背景、已有世界书、正文、大纲、角色与力量体系，列出不可违背的既有约束。
2. 解释该主题如何影响自然规则、社会结构、资源分配、权力体系、日常生活、伦理观念和主要矛盾。
3. 每条强大能力、技术、制度或资源必须有成本、限制、反制方式和失败后果。
4. 世界观要能够产生人物选择和剧情冲突，避免只做百科式设定堆砌。
5. 输出：核心概念、基础规则、历史成因、地理与环境、势力结构、经济与资源、文化与禁忌、力量/技术体系、普通人的生活、5 个剧情钩子、与现有设定的冲突检查。
6. 区分已确认设定、建议设定和待确认问题。
7. 用户确认前不写入项目；确认后按主题拆分写入 `.storydex/worldbook/`，不得覆盖无关条目。

主题：[主题]
题材：[奇幻/科幻/都市/历史/悬疑/自定义]
世界规模：[单城/国家/大陆/星球/多世界]
氛围关键词：[关键词]
```
