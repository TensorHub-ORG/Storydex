# 制定卷纲与章节大纲

> 根据当前故事状态规划目标、冲突、转折、伏笔和章节推进节奏。

## 可直接使用的指令

```prompt
请基于当前 Storydex 小说项目，为[卷名或剧情阶段]制定卷纲与章节大纲。

要求：
1. 读取现有正文、角色状态、未解决线索、世界观、剧本和章节进度，先总结故事当前起点。
2. 明确本卷的主目标、主要对手/阻力、人物弧线、情感变化、核心悬念和卷末状态变化。
3. 每章至少包含：章节目标、冲突、关键行动、信息揭示、情绪变化、结尾钩子、涉及角色、需要保持的连续性。
4. 伏笔要注明埋设、强化和回收位置；不得提前使用角色尚未知晓的信息。
5. 控制升级节奏，避免连续重复同类冲突或仅靠巧合推进。
6. 输出总纲、阶段拆分、章节表、伏笔表、风险与一致性检查。
7. 只输出大纲草案；用户确认后再写入 `.storydex/scripts/`，不直接修改正文。

计划章节数：[章节数]
本卷目标字数：[字数，可留空]
节奏偏好：[快/中/慢]
必须包含的事件：[事件，可留空]
```
