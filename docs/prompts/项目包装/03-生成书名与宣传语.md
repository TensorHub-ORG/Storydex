# 生成书名、短标题与宣传语

> 根据当前项目的核心冲突、意象和题材定位，生成可筛选的命名方案。

## 可直接使用的指令

```prompt
请读取当前 Storydex 小说项目，生成书名和宣传语候选。

要求：
1. 以正文、项目简介、大纲、世界观和主角动机为依据，不使用与项目无关的热门词拼接。
2. 避免直接泄露结局和核心谜底，避免与现有角色、地点、组织名称混淆。
3. 输出 3 组方向，每组包含：5 个书名、每个书名的命名逻辑、1 条 20 字以内短宣传语、1 条 60 字以内长宣传语。
4. 三组方向分别侧重：[核心冲突]、[人物情感]、[世界观意象]；如用户提供其他方向则替换。
5. 最后给出综合推荐前 3 名及理由。
6. 只输出候选方案，不重命名项目或文件。

题材定位：[题材]
命名偏好：[简洁/诗性/强情节/商业化/自定义]
禁用词：[可留空]
```
