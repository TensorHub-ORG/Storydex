# 根据当前小说项目生成封面 AI 绘图提示词

> 从项目题材、主角、场景和视觉意象中提炼可直接用于绘图模型的封面提示词。

## 可直接使用的指令

```prompt
请基于当前 Storydex 小说项目，为小说封面生成 AI 绘图提示词。

执行要求：
1. 先读取项目简介、现有正文、主角档案、关键地点、世界书和大纲，确认题材、时代、氛围与主视觉元素。
2. 不得添加与项目冲突的服装、种族、科技水平、建筑风格或角色外貌；缺失细节应标记为可选设计，不要伪装成原设定。
3. 封面构图必须预留书名和作者名区域，主体在缩略图尺寸下仍可识别。
4. 避免直接生成带文字的画面内容，文字排版由后期完成。
5. 分别输出：
   - 视觉概念摘要
   - 中文正向提示词
   - 英文正向提示词
   - 负向提示词
   - 构图、镜头、光线、色彩和材质建议
   - 3 个风格变体
6. 只输出提示词，不修改项目文件。

绘图模型：[Midjourney / Stable Diffusion / DALL·E / 通用]
画幅比例：[默认2:3竖版]
目标风格：[写实/插画/国风/动漫/电影概念设计/自定义]
```
