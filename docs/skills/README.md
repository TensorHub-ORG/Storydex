# Storydex 内置技能模板

本目录是新建和初始化 Storydex 小说项目时，`.storydex/.agent/skills/` 内置技能文档的唯一来源。桌面版会把本目录封装进安装包，后端通过 `STORYDEX_BUILTIN_SKILLS_ROOT` 定位并读取。

## 模板要求

- 所有技能必须适用于任意小说项目，不得包含固定项目名、角色名或世界观事实。
- 每个技能必须包含用途、触发条件、输入、证据规则、资产落点、执行步骤、输出模板、自检和安全边界。
- 缺少项目证据的信息必须标记为“未知”“候选”或“待确认”。
- 用户自定义过的项目技能不会被自动覆盖；仍等于旧版默认内容的技能可以安全升级。
- `story_preset_constraints.md` 是动态技能底稿，项目激活预设会附加在其“当前激活预设”章节。
- `registry.json` 由程序生成，负责登记技能 ID、意图和资产落点，不属于 Markdown 技能正文。

## 项目初始化行为

1. Storydex 读取本目录全部 `.md` 文件，跳过 `README.md`。
2. 将模板复制到当前项目 `.storydex/.agent/skills/`。
3. 已存在且被用户修改的同名技能保持不变。
4. 旧版默认技能自动迁移为本目录的新模板。
5. 动态预设技能根据当前项目激活预设重新生成。
