# Storydex v0.2.0 Release Candidate

本版本聚焦桌面版更新能力、Coomi 交互体验、预设管理与知识图谱能力，并修复 Markdown 文件链接在应用内跳转异常的问题。

## 主要更新

- 新增桌面版差分更新能力：基于 `electron-updater` 与 NSIS blockmap，支持应用内检查、增量下载、重启安装。
- 新增系统设置中的“更新与关于”入口，可查看当前版本、检查更新、查看下载进度并安装已下载版本。
- 修复 Markdown 预览中的相对文件链接：点击 `001.md` / `002.md` 等链接时会在中间文件区域打开对应文件，不再跳到异常登录地址或白屏。
- 优化 Coomi 多问题审批：后端一次性发送多个问题，前端支持题目切换、草稿保存、统一提交和收起面板。
- 优化本轮修改提交提示：展示变更文件列表，并允许暂时收起提示面板。
- 优化预设编辑器采样参数：`temperature`、`top_p`、`frequency_penalty`、`presence_penalty` 支持滑块和数值输入双向编辑。

## 相比 v0.1.1 的累计变化

- 包含当前 `main` 已有的知识图谱与预设管理增强。
- 增强 SillyTavern 预设导入、宏运行时、预设编译与运行时上下文相关能力。
- 增强 Agent Git 自动提交、模型拉取与预设上下文菜单相关流程。
- 增强故事 WIKI / 知识图谱服务及相关测试覆盖。

## 下载内容

- `StorydexSetup-x64-0.2.0.exe`：Windows x64 安装包。
- `StorydexSetup-x64-0.2.0.exe.blockmap`：安装包差分更新索引。
- `latest.yml`：自动更新元数据。
- `Storydex-win-unpacked.zip`：免安装可执行目录压缩包。
- `SHA256SUMS.txt`：发行文件校验值。

## 发布前检查

- 确认更新服务器同一目录包含新版 `.exe`、`.exe.blockmap`、`latest.yml`。
- 确认更新服务器保留旧版 `.exe.blockmap`，否则客户端会回退为全量下载。
- 确认打包版“系统设置 -> 更新与关于 -> 检查更新 / 下载更新（增量） / 重启并安装”流程可用。
- 确认 Markdown 预览中点击相对 `.md` 链接会在中间文件区域打开目标文件。
