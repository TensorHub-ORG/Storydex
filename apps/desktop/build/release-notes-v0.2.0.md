# Storydex v0.2.0 Windows 正式版

Storydex v0.2.0 现已发布。本版本重点改进桌面版更新体验、Coomi 交互、预设管理、知识图谱 / WIKI 能力，并修复 Markdown 文件链接在应用内打开异常的问题。

建议所有 v0.1.x 用户升级到此版本。

## 新增功能

- 新增桌面版应用内更新能力，后续版本可在“系统设置 -> 更新与关于”中检查、下载并安装更新。
- 新增更新下载进度展示，应用会显示当前版本、可用版本、更新状态和下载进度。
- 新增 Coomi 多问题审批面板，多个问题可以集中展示、逐项切换、保存草稿并统一提交。
- 新增本轮修改提交提示中的变更文件列表，更容易确认 Coomi 本轮实际改动了哪些文件。

## 修复

- 修复 Markdown 预览中的相对文件链接问题。点击 `001.md`、`002.md` 等蓝色链接时，现在会在中间文件区域打开对应文件，不再跳到异常地址或白屏。
- 修复 Markdown 文件链接点击后难以返回主界面的问题。

## 改进

- 优化 Coomi 待回答问题和提交提示面板，支持暂时收起，减少对输入区的遮挡。
- 优化预设编辑器采样参数，`temperature`、`top_p`、`frequency_penalty`、`presence_penalty` 支持滑块和数值输入双向编辑。
- 增强外部预设导入、宏运行时、预设编译和运行时上下文适配。
- 增强知识图谱 / WIKI 的故事状态、关系和资料组织能力。
- 增强 Agent 模型拉取、预设上下文、Git 自动提交与提交摘要相关流程。

## 下载内容

- `StorydexSetup-x64-0.2.0.exe`：Windows x64 安装包，推荐大多数用户下载。
- `Storydex-win-unpacked.zip`：免安装版本，解压后运行 `Storydex.exe`。
- `SHA256SUMS.txt`：发布文件校验值。

以下文件用于应用内更新，不需要手动打开：

- `latest.yml`
- `StorydexSetup-x64-0.2.0.exe.blockmap`

## 升级说明

- 从旧版本升级时，可以直接运行新版安装包覆盖安装。
- Storydex 不会主动删除用户项目；升级前仍建议确认重要写作项目已经备份或提交到版本控制。
- 如果使用免安装版本，请解压到新的目录后运行 `Storydex.exe`。
