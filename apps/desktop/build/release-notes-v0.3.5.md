# Storydex v0.3.5

## 大文件与工作区体验

- 小于 2MB 的文件保持完整编辑，2～20MB 文件渐进加载，大于 20MB 默认进入只读快速预览。
- 首屏按约 256KB 分块读取，支持 UTF-8 字符边界、BOM、后台行索引、LRU 缓存、快速跳转和过期请求取消。
- 资源浏览器空目录提示更紧凑，并新增统一问题面板与错误、警告、迁移提示分级。

## 长期记忆与变量

- `.storydex/memory` 只保存正式变量、事实、人物状态、关系、时间线等长期记忆，禁止写入历史会话、工具日志和临时文件。
- 历史会话继续保存在 `.storydex/.agent/sessions`；`.storydex/temp` 仅作为普通临时工作台，不参与索引、诊断、自动清理或常规上下文注入。
- 引入自适应模块目录、schemaVersion、稳定 ID、revision、变更账本、恢复点和非破坏性旧项目迁移。

## 诊断、依赖与安装

- UTF-8 BOM 可兼容读取并一键移除；诊断标记与 Git 修改状态彻底分离。
- 开发、CI 和安装包内嵌环境严格固定并验证 `coomi-agent==1.1.2`。
- 差分安装使用独立可见的辅助窗口，展示安装进度、失败恢复和日志；安装锁阻止半更新状态启动，完成后由用户选择是否启动 Storydex。

## 安装方式

已安装 v0.3.4 的用户可在 Storydex 内检查并下载 v0.3.5。安装包与便携包继续内置可迁移 Python 3.9 环境、完整后端依赖和 MinGit，无需额外安装 Python 或 Git。
